# Simple FFT Visualizer

画像のFFT（高速フーリエ変換）を可視化し、ローパス・ハイパスフィルタの効果を直感的に理解できるツールです。

## 機能

- 画像の周波数スペクトラムを表示
- 3つのフィルタモード（Original / LowPass / HighPass）
- フィルタ半径の可視化（スペクトラム上に円で表示）
- リアルタイムでフィルタリング結果を確認
- **4K画像の自動FHDリサイズ**（処理速度向上のため）
- **表示用と処理用の両方でFHD最適化**
- ウィンドウサイズの自動調整

## 使用方法

### 基本的な使用方法

```bash
python tools/fft_visualizer.py <画像ファイルのパス>
```

### 例

```bash
# 画像ファイルを指定して実行
python tools/fft_visualizer.py sample.jpg

# 相対パスでも可能
python tools/fft_visualizer.py ../images/test.png

# 絶対パスでも可能
python tools/fft_visualizer.py /path/to/your/image.jpg

# 4K画像も軽快に動作（自動的にFHDにリサイズ）
python tools/fft_visualizer.py 4k_image.jpg
```

### オプション

```bash
# ヘルプを表示
python tools/fft_visualizer.py --help
```

## 操作方法

1. **ツール起動**: 画像ファイルを指定してツールを起動
2. **フィルタ切り替え**: 'f'キーでフィルタモードを切り替え
3. **半径調整**: **'+'/'−'キー**でフィルタ半径を調整
4. **結果確認**: 右側でフィルタリングされた画像を確認
5. **終了**: 'q'キーを押すかウィンドウを閉じる

### キーボードショートカット

- **'f'**: フィルタモード切り替え
  - **Original**: 元画像をそのまま表示
  - **LowPass**: 低周波のみ通す（ぼやけた画像、主要構造）
  - **HighPass**: 高周波のみ通す（エッジ、細かいディテール）
- **'+'**: フィルタ半径を増加（より多くの周波数を通す）
- **'−'**: フィルタ半径を減少（より少ない周波数を通す）
- **'q'**: ツール終了

**注意**: 矢印キーは使用しません。確実に動作する'+'/'−'キーを使用してください。

## 画面の見方

- **左側**: 周波数スペクトラム（明るい部分ほど強い周波数成分）
  - **緑の円**: ローパスフィルタの通過範囲（PASS）
  - **赤の円**: ハイパスフィルタのカット範囲（CUT）
- **右側**: フィルタリングされた画像
  - **Original**: 元画像
  - **LowPass**: 低周波成分のみで復元された画像（ぼやけた画像）
  - **HighPass**: 高周波成分のみで復元された画像（エッジ強調）
- **上部**: フィルタモードと半径の情報
- **下部**: 操作説明

## 技術的な詳細

### FFT処理の流れ

1. 画像をグレースケールで読み込み
2. **FHD（1920x1080）を超える場合は自動リサイズ**（処理速度向上）
3. 2次元FFTを実行
4. 周波数成分を中央にシフト（fftshift）
5. 対数スケールで振幅スペクトラムを計算
6. 表示用に正規化

### フィルタリング機能

- **ローパスフィルタ**: 中心から指定半径内の低周波成分のみ通す
- **ハイパスフィルタ**: 中心から指定半径外の高周波成分のみ通す
- **可視化**: スペクトラム上に円でフィルタ範囲を表示
- **リアルタイム更新**: 半径変更時に即座に結果を反映

### 画像リサイズ機能（パフォーマンス最適化）

- **処理用リサイズ**: FHD（1920x1080）を超える画像を自動リサイズ
- **表示用リサイズ**: 結合画像がFHDを超える場合も追加でリサイズ
- **リサイズ方法**: アスペクト比を保持してINTER_AREA補間
- **効果**: 4K画像でも軽快に動作
- **ウィンドウ調整**: FHDサイズに収まるように自動調整

### パフォーマンス改善

- **4K → FHD変換**: 処理時間を大幅短縮
- **メモリ使用量削減**: 大容量画像でも安定動作
- **リアルタイム性**: フィルタ切り替えが即座に反映

## トラブルシューティング

### よくある問題

1. **画像が読み込めない**
   - ファイルパスが正しいか確認
   - 対応形式の画像ファイルか確認
   - ファイルの読み取り権限があるか確認

2. **ウィンドウが表示されない**
   - OpenCVが正しくインストールされているか確認
   - GUI環境で実行しているか確認

3. **キー操作が効かない**
   - ウィンドウがアクティブ（選択状態）になっているか確認
   - '+'/'−'キーを使用（矢印キーは非対応）

4. **動作が重い**
   - 4K以上の画像は自動的にFHDサイズにリサイズされます
   - それでも重い場合は、より小さな画像を使用してください

## 例

### 基本的な使用例

```bash
# 画像ファイルを指定して実行
python tools/fft_visualizer.py sample.jpg

# 4K画像も自動的にFHDにリサイズされて軽快に動作
python tools/fft_visualizer.py 4k_image.png

# QHD画像も自動リサイズ
python tools/fft_visualizer.py 2560x1440_image.jpg
```

### フィルタ効果の確認

1. **ローパスフィルタ**: 'f'キーでLowPassに切り替え
   - '−'キーで半径を小さくすると、より強くぼかされる
   - 対象物の大まかな形状のみが残る

2. **ハイパスフィルタ**: 'f'キーでHighPassに切り替え
   - '+'キーで半径を大きくすると、より細かいエッジまで抽出
   - 境界やテクスチャが強調される

### テスト用画像の作成

```bash
# テスト画像作成スクリプトを実行
python tools/create_test_images.py
```

縞模様やチェッカーボードなどの画像で、周波数フィルタの効果を明確に確認できます。

### 高解像度画像での使用

```bash
# 高解像度画像も軽快に動作
python tools/fft_visualizer.py tools/test_images/high_resolution.png
```

QHD（2560x1440）の高解像度テスト画像も自動的にFHDにリサイズされて処理されます。
