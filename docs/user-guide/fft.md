# FFT ビジュアライザー (`pochi fft`)

`pochi fft` は単一画像の FFT (高速フーリエ変換) スペクトルを可視化する. 周波数特徴の分布を視覚的に確認し, FFT 特徴量抽出器のパラメータ調整に利用する.

## 1. 基本コマンド

```bash
uv run pochi fft --input image.png
```

## 2. 引数

| 引数 | 短縮形 | 説明 |
|------|--------|------|
| `--input` | `-i` | 入力画像パス (必須) |

## 3. 表示内容

- 元画像 (グレースケール変換後)
- 振幅スペクトル (対数スケール)
- 位相スペクトル
- 半径方向プロファイル

## 4. 関連リファレンス

- 抽出される特徴量とパラメータの詳細は [FFT 特徴量](features/fft.md) を参照.
