# 画像集約 (`pochi aggregate`)

`pochi aggregate` は複数のサブディレクトリに散らばった画像を 1 つのディレクトリにまとめる. 学習データセット作成や手動アノテーション前の集約に使用する.

## 1. 基本コマンド

```bash
# 画像をコピーして集約 (デフォルト)
uv run pochi aggregate --input ./data

# 画像を移動して集約 (元ファイルは削除される)
uv run pochi aggregate --input ./data --mode move
```

## 2. 引数

| 引数 | 短縮形 | 説明 |
|------|--------|------|
| `--input` | `-i` | 入力ディレクトリ (必須) |
| `--mode` | `-m` | 集約モード: `copy` または `move` (デフォルト: `copy`) |

## 3. 動作

入力ディレクトリ配下を再帰的にスキャンし, 画像ファイル (`.jpg`, `.png`, `.bmp` 等) を入力ディレクトリ直下の集約先にまとめる. ファイル名衝突時は連番を付与して上書きを防ぐ.
