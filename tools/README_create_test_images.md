# Create Test Images

FFT Visualizer用のテスト画像を自動生成するスクリプトです。様々な周波数特性を持つパターン画像を作成し、FFTの効果を視覚的に確認できます。

## 概要

このスクリプトは、FFT（高速フーリエ変換）の学習や検証に適した9種類のテスト画像を自動生成します。各画像は異なる周波数特性を持ち、ローパス・ハイパスフィルタの効果を明確に観察できるように設計されています。

## 機能

- **9種類のテスト画像を自動生成**
- **周波数特性の異なるパターン**（縞模様、チェッカーボード、正弦波など）
- **高解像度テスト画像**（リサイズ機能の検証用）
- **PNG形式での出力**（ロスレス圧縮）
- **FFT Visualizerとの連携**

## 生成される画像

### 1. 縦縞模様 (vertical_stripes.png)
- **サイズ**: 256x256
- **特徴**: 垂直方向の縞模様
- **周波数特性**: 水平方向に強い周波数成分
- **FFT効果**: ローパスで縞がぼやけ、ハイパスで境界が強調

### 2. 横縞模様 (horizontal_stripes.png)
- **サイズ**: 256x256
- **特徴**: 水平方向の縞模様
- **周波数特性**: 垂直方向に強い周波数成分
- **FFT効果**: 縦縞とは異なる周波数分布を確認可能

### 3. チェッカーボード (checkerboard.png)
- **サイズ**: 256x256
- **特徴**: 32x32ピクセルの市松模様
- **周波数特性**: 水平・垂直両方向に周期的成分
- **FFT効果**: 格子状の周波数パターンが観察可能

### 4. 円形パターン (circle.png)
- **サイズ**: 256x256
- **特徴**: 中央に白い円（半径80ピクセル）
- **周波数特性**: 低周波成分が主体
- **FFT効果**: ローパスで形状保持、ハイパスで境界のみ抽出

### 5. 斜め縞模様 (diagonal_stripes.png)
- **サイズ**: 256x256
- **特徴**: 45度方向の斜め縞
- **周波数特性**: 斜め方向の周波数成分
- **FFT効果**: 回転した周波数分布を確認

### 6. ガウシアンノイズ (gaussian_noise.png)
- **サイズ**: 256x256
- **特徴**: 平均128、標準偏差50のランダムノイズ
- **周波数特性**: 全周波数に均等に分布
- **FFT効果**: フィルタの周波数選択性を明確に確認

### 7. 正弦波パターン (sine_wave.png)
- **サイズ**: 256x256
- **特徴**: 水平方向の正弦波（周期32ピクセル）
- **周波数特性**: 特定周波数に鋭いピーク
- **FFT効果**: 単一周波数成分の抽出・除去を確認

### 8. 複合パターン (complex_pattern.png)
- **サイズ**: 256x256
- **特徴**: 複数の正弦波を合成
  - 高周波成分（周期16ピクセル）
  - 低周波成分（周期64ピクセル）
  - 垂直方向成分（周期32ピクセル）
- **周波数特性**: 複数の周波数成分が混在
- **FFT効果**: 選択的な周波数除去・抽出の効果を確認

### 9. 高解像度テスト画像 (high_resolution.png)
- **サイズ**: 2560x1440（QHD解像度）
- **特徴**: 高解像度の縦縞模様
- **用途**: リサイズ機能のテスト
- **FFT効果**: 大画像でのパフォーマンス確認

## FFT Visualizerとの連携

### 基本的な使用例

```bash
# テスト画像を生成
python tools/create_test_images.py

# 各画像でFFT効果を確認
python tools/fft_visualizer.py tools/test_images/vertical_stripes.png
python tools/fft_visualizer.py tools/test_images/checkerboard.png
python tools/fft_visualizer.py tools/test_images/sine_wave.png
```

### 学習用の推奨順序

1. **円形パターン**: 基本的なローパス・ハイパス効果を理解
2. **縦縞・横縞**: 方向性のある周波数成分を確認
3. **正弦波**: 単一周波数の抽出・除去を観察
4. **チェッカーボード**: 複雑なパターンでの効果を確認
5. **複合パターン**: 複数周波数成分の選択的処理を学習
6. **ガウシアンノイズ**: フィルタの周波数選択性を確認

## 技術的な詳細

### 画像生成アルゴリズム

#### 縞模様の生成
```python
# 20ピクセル周期の縞模様
for i in range(0, 256, 20):
    img[:, i:i+10] = 255  # 10ピクセル幅の白い縞
```

#### 正弦波の生成
```python
# 32ピクセル周期の正弦波
for j in range(256):
    value = 128 + 127 * np.sin(2 * np.pi * j / 32)
    img[i, j] = int(value)
```

#### 複合パターンの生成
```python
# 複数の周波数成分を合成
value = 128  # ベース値
value += 50 * np.sin(2 * np.pi * j / 16)   # 高周波
value += 30 * np.sin(2 * np.pi * j / 64)   # 低周波
value += 20 * np.sin(2 * np.pi * i / 32)   # 垂直成分
```

### ファイル出力

- **形式**: PNG（ロスレス圧縮）
- **ビット深度**: 8bit グレースケール
- **出力先**: `tools/test_images/` ディレクトリ
- **命名規則**: 英語の説明的な名前

## カスタマイズ

### パラメータの変更

スクリプトを編集することで、以下のパラメータを変更できます：

```python
# 画像サイズの変更
img = np.zeros((512, 512), dtype=np.uint8)  # 512x512に変更

# 縞模様の周期変更
for i in range(0, 256, 10):  # 10ピクセル周期に変更
    img[:, i:i+5] = 255

# 正弦波の周期変更
value = 128 + 127 * np.sin(2 * np.pi * j / 16)  # 16ピクセル周期
```

### 新しいパターンの追加

```python
# 新しいパターンの例：同心円
def create_concentric_circles():
    img = np.zeros((256, 256), dtype=np.uint8)
    center = (128, 128)
    for radius in range(20, 120, 20):
        cv2.circle(img, center, radius, 255, 2)
    return img
```

## トラブルシューティング

### よくある問題

1. **ディレクトリが作成されない**
   - 書き込み権限があるか確認
   - パスが正しいか確認

2. **画像が生成されない**
   - OpenCVが正しくインストールされているか確認
   - ディスク容量が十分にあるか確認

3. **FFT Visualizerで読み込めない**
   - 画像ファイルが正常に生成されているか確認
   - ファイルパスが正しいか確認
