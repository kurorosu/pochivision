# データ概要表示CLI アプリケーション

CSVファイルの基本情報を素早く確認し、美しいヒストグラムを表示できるインタラクティブなコマンドラインツールです。

## 特徴

### 🎯 主要機能
- **CSVファイル読み込み**: pandas を使用した高速データ読み込み
- **データ概要表示**: 基本統計情報、データ型分布、列詳細の美しい表示
- **インタラクティブヒストグラム**: ターミナル内でのリアルタイムグラフ表示
- **クラス別色分け**: カテゴリデータによる自動色分けヒストグラム

### 🎨 ユーザーインターフェース
- **矢印キー操作**: 直感的なメニューナビゲーション
- **美しい表示**: `rich` ライブラリによる色付きテーブルとパネル
- **リアルタイムグラフ**: `plotext` によるターミナル内グラフ表示
- **プログレスバー**: データ読み込み時の視覚的フィードバック

## インストール

### 前提条件
- Python 3.8+
- conda または pip

### 依存関係のインストール

```bash
# condaを使用する場合
conda install --file requirements.txt -c conda-forge

# pipを使用する場合
pip install -r requirements.txt
```

### 主要依存関係
- `pandas`: データ処理
- `numpy`: 数値計算
- `rich`: 美しいターミナル表示
- `plotext`: ターミナル内グラフ表示
- `questionary`: インタラクティブメニュー
- `click`: コマンドライン引数処理

## 使用方法

### 基本的な使用

```bash
# ツールを起動
python tools/feature_selector_cli.py

# またはモジュールとして実行
python -m tools.feature_selector_cli
```

### メニュー操作

1. **矢印キー（↑↓）** でメニュー項目を移動
2. **Enterキー** で選択を確定
3. **Ctrl+C** でいつでも終了可能

## クラス構造

### `DataOverviewCLI`
メインアプリケーションクラス

#### 主要メソッド
- `run()`: アプリケーションのメインループ
- `_load_csv_file()`: CSVファイル読み込み
- `_show_data_overview()`: データ概要表示
- `_show_histogram()`: ヒストグラム表示制御
- `_show_simple_histogram()`: 単純ヒストグラム表示
- `_show_class_histogram()`: クラス別色分けヒストグラム表示

## 使用例

### 1. 基本的なデータ探索

```bash
python tools/feature_selector_cli.py
```

1. 「1. CSVファイルを読み込む」を選択
2. ファイルパスを入力: `./data/features.csv`
3. 「2. データの概要を表示」でデータの全体像を確認

### 2. 特徴量分布の確認

1. 「3. ヒストグラムを表示」を選択
2. 矢印キーで特徴量を選択（例: `brightness_mean`）
3. 「色分けしない」を選択して単純な分布を確認

### 3. クラス別比較

1. 「3. ヒストグラムを表示」を選択
2. 矢印キーで特徴量を選択
3. クラス列（例: `class`）を選択してクラス別分布を比較

## 注意事項

### ファイル形式
- CSV形式のファイルのみサポート
- UTF-8エンコーディング推奨
- ヘッダー行必須

### データ制限
- クラス数が10個を超える場合は最初の10個のみ表示
- 欠損値は自動的に除外してヒストグラム作成
- 数値列のみヒストグラム表示可能

### パフォーマンス
- 大きなファイル（100MB+）の場合は読み込みに時間がかかる場合があります
- メモリ使用量はファイルサイズに依存

## トラブルシューティング

### よくあるエラー

#### ファイルが見つからない
```
ファイルが見つかりません: /path/to/file.csv
```
- ファイルパスが正しいか確認
- ファイルの存在を確認
- 相対パス/絶対パスの確認

#### 数値列が見つからない
```
数値列が見つかりません。
```
- CSVファイルに数値データが含まれているか確認
- データ型が正しく認識されているか「2. データの概要を表示」で確認

#### メモリエラー
- ファイルサイズを小さくする
- 不要な列を事前に削除
- より多くのメモリを持つ環境で実行
